Detailed Description

A rectangle with special properties.

It automatically stores all important points of a rectangle (topLeft, topRight, center...), making everything easier.

Note
Since everything's public, take care to not modify the attributes directly. Use the predefined functions to alter things!

Zendef:

Note
NEVER NEVER try to modify x/y/w/h directly! Always do it through the add()/set() methods! That's because this class needs to refresh internal points and stuff.

◆ placeOnTop()

void Rectangle::placeOnTop ( Rectangle other)

Places this rectangle immediately above other.

Note
It only considers the y axis, x is left unchanged.

For developers

The only purpose of this method is to make possible for the player to stay above a platform. Say if it's collision Rectangle passes through the platform, we place it above it with this method.
