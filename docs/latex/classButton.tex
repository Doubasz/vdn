\doxysubsection{Detailed Description}
A button class to represent an interactive button with SDL2

This class provides functionality for creating, rendering, and interacting with a button in a graphical user interface. It supports multiple states (normal, hover, pressed, disabled), customizable colors, and can handle click events through a callback.

\begin{DoxyNote}{Note}
The button uses a rectangle ({\ttfamily \doxylink{structRectangle}{Rectangle}}) for its positioning and a text ({\ttfamily std\+::string}) for its label, with support for font rendering using SDL\+\_\+ttf. 
\end{DoxyNote}
