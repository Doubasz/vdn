\doxysubsection{Detailed Description}
Base class representing any game object with a position, size, health, and physics.

This class defines a rectangular-\/shaped entity with position, velocity, and collision logic. Entities can also take damage, experience knockback, and be toggled as visible or invisible.\hypertarget{classEntity_autotoc_md0}{}\doxysubsubsection{\texorpdfstring{How to use}{How to use}}\label{classEntity_autotoc_md0}
