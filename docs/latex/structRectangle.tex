\doxysubsection{Detailed Description}
A rectangle with special properties.

It automatically stores all important points of a rectangle (top\+Left, top\+Right, center...), making everything easier.

\begin{DoxyNote}{Note}
Since everything\textquotesingle{}s public, take care to not modify the attributes directly. Use the predefined functions to alter things!
\end{DoxyNote}
Zendef\+: \begin{DoxyNote}{Note}
NEVER NEVER try to modify {\ttfamily x/y/w/h} directly! Always do it through the {\ttfamily add()}/{\ttfamily set()} methods! That\textquotesingle{}s because this class needs to refresh internal points and stuff. 
\end{DoxyNote}
